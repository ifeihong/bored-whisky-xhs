# BORED WHISKY 小红书爆款图文生成 Skill

读取用户输入 → 一问一答收集信息（或智能推荐）→ 按6+2模板体系+风格库生成图片提示词 + 小红书文案。
**重要：一次只问一个问题，等用户回答后再问下一个。**
如果用户已经在当前消息提供了某项信息，直接记录并跳过对应问题。
所有选择题都必须做成编号选项，不要用开放式问题收集风格、情绪、模板偏好。
每次生成必须与之前的内容有明显差异（不同模板、不同画风、不同色彩、不同装饰、不同文案角度、不同热梗嫁接方式）。

**品牌知识库**详见同目录下的 `brand.md`，包含：品牌DNA、用户画像、使用场景、核心词汇、视觉品牌元素、瓶身比例规则、LOGO植入规则、IP限制规则。生成时必须遵守 `brand.md` 中的所有规定。

**第0问：输入主题（或AI随机）**

3. **参考图复刻模式** — 上传参考图，AI分析并复刻为BORED WHISKY风格

用户选1进入第2问。用户选2跳过第2问，直接进入第3问（AI自动选择模板和画风）。用户选3要求上传参考图，AI分析参考图风格后确认，再进入第3问。

## 通用视觉规范（所有输出遵守）

**图片生成模型适配：** GPT Image2 / Nano Banana Pro / Seedream5.0
**比例：** 3:4 竖版（小红书标准，1024×1365px）

### 品牌元素

- **BORED WHISKY 产品瓶身（参考图1）** — 产品角度/方向/可见细节必须严格按照参考图1还原，不得出现参考图1中不存在的角度、方向或设计细节。产品瓶身上自带的 BAYC #480 形象自然保留即可。
- **BORED WHISKY Logo（参考图2）** — 必须植入画面角落（右下角/左上角/底部居中），Logo 文字保持单行不折行，大小占画面宽度约8-15%，清晰可辨识。
- **瓶身比例：** 约等于成年人手掌大小（15-18cm），与画面中其他参照物保持真实比例（见 `brand.md`）。

### 猿猴/猴子规则（红线，不可违反）

**除了产品瓶身画面外，严禁其他画面中出现任何猿猴/猴子/无聊猿形象：**
- 不要在图片中额外添加任何猴子、猿猴、无聊猿的独立形象、剪影、插画或贴纸
- 产品瓶身上自带的 BAYC #480 形象可以保留，但不要将其单独提取、放大或作为独立元素使用
- 不要生成"穿西装的猿猴剪影""猿猴侧影""猿猴坐姿"等任何猿猴形象
- 提示词中不要出现"无聊猿形象""猿猴剪影""猿猴元素"等描述
- 装饰元素库中所有动物相关元素仅限猫、狗、鸟类等普通动物，严禁任何灵长类动物

### 人物处理规范（规避AI审核）

- 人物**不露完整正脸**
- 优先方案：侧脸/背影/局部（手部/肩部）
- 可用道具遮挡：太阳眼镜、威尼斯面具、潮流口罩、帽子压低、书本/酒杯挡脸
- 可用景深处理：脸部适度虚化，焦点在产品上
- 可用创意方式：剪影、倒影、镜中像、局部特写、前景遮挡
- **禁止：** 清晰完整的正面人脸特写

- **文字装饰：** 从「文字装饰样式库」中选择1-3种装饰
- **副标题/文案：** 简洁无衬线体，与标题形成粗细对比

- 必须有**设计感**和**精致感**，不能是普通、平庸、模板化的图片
- 构图要有创意和巧思，避免呆板的产品居中+纯色背景+无装饰
- 色彩搭配要有层次和质感，避免廉价感或过于饱和的塑料感
- 光影要有氛围感，避免平面化、无立体感的生硬图片
- 装饰元素要与画风统一，不要混搭冲突（如S9极简风不要贴满Y2K贴纸）
- 整体风格参考：高端生活方式杂志、潮流品牌视觉、精致露营美学
- 在提示词中明确加入"精致设计感""有创意构图""高级质感"等引导词

## 提示词写作规范

1. **先写画面，再写风格**：先描述主体位置和比例，再用自然语言描述画风特征、色彩、构图
2. **参考图指代明确**：直接写"参考图1/参考图2"，不要复述图片细节
3. **风格维度必写（核心！）**：每个提示词必须明确包含6个风格维度，但**必须用自然语言描述，禁止在提示词正文中出现S/C/A编号**：
   - 画风：描述视觉特征（如"复古胶片质感，有颗粒感和暖黄偏色"），不是"S4复古胶片"
   - 色彩：描述具体颜色（如"暖黄橙棕为主，米色辅色，深红墨绿点缀"），不是"C3复古胶片"
   - 角度：描述拍摄角度（如"从人物身后过肩视角"），不是"A8过肩视角"
   - 背景纹理：描述材质（如"深色橡木纹理桌面"），不是"木质桌面"
   - 装饰元素：描述具体物件（如"金色散景光斑、和纸胶带斜贴"），不是"光斑、胶带"
   - 滤镜效果：描述效果（如"细微胶片颗粒+四角暗角"），不是"胶片颗粒+暗角"
4. **编号仅用于风格参数记录**：S/C/A编号只写在输出的「风格参数记录」部分，绝对不能混入给生图模型的提示词正文中
5. **色号/比例写死**：所有比例明确写"3:4竖版"，颜色用具体色值或明确描述
6. **人物处理优先**：涉及人物时，首先说明不露正脸的处理方式
7. **品牌元素必提**：每个提示词都必须明确包含参考图1（产品）和参考图2（Logo），缺一不可。Logo 必须有具体位置描述，Logo 文字保持单行不折行
8. **视觉风格多样化**：每次生成时，艺术画风、色彩方案、装饰元素组合必须与最近2次明显不同
9. **情绪一致**：提示词的情绪词和氛围必须与第3问选择的情绪基调一致
10. **画风-色彩-情绪匹配**：参考「情绪-画风-色彩搭配建议」，避免违和搭配（如忧郁情绪配Y2K+荧光撞色）
11. **装饰元素适配画风**：极简风只选1-2个极简装饰；潮流风可选4-6个丰富装饰；治愈风选2-3个温柔装饰
12. **细节丰富度要求（核心）**：提示词必须包含以下至少4个维度的细节描述：
    - **光影**：光源方向、明暗对比、高光位置、阴影层次、氛围光
    - **材质质感**：玻璃反光、酒液通透感、背景纹理、搭配元素材质
    - **景深层次**：前景/中景/背景关系、虚化程度、空间纵深感
    - **色调氛围**：具体色彩描述、饱和度、明暗调性
    - **文字排版**：字体类型、描边/阴影、错落角度、装饰样式
    - **画风特征**：所选艺术画风的标志性视觉元素
13. **避免：**
    - 不要极简留白过度（小红书需要信息密度和设计感，但S9极简风除外）
    - 不要细字体（标题必须醒目粗体，但S2/S9极简风可用细体）
    - 不要低对比浅灰字（必须清晰可读）
    - 不要清晰完整正脸（规避审核风险）
    - 不要让标题和装饰遮挡产品标签
    - 不要全息投影、数据流、粒子特效（不符合品牌调性，S10赛博朋克除外但需克制）
    - 不要平庸无设计感的图片（产品居中+纯色背景+无装饰=不合格）
    - 不要廉价感、塑料感、过度饱和的色彩
    - 不要混搭冲突风格（如法式慵懒+赛博朋克霓虹）
    - 不要装饰元素过多过杂（最多6个，且必须与画风统一）
    - 不要猿猴/猴子独立形象（产品瓶身自带的除外）
    - 不要在文案中出现具体容量数字（200ml/350ML等）
