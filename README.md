# BORED WHISKY 小红书图文生成 Skill

> **BORED WHISKY（年轻人的口袋威士忌）** 专属小红书图文生成器。通过一问一答交互，自动输出 **AI 文生图提示词** + **小红书文案（标题 / 正文 / 标签）**，适配 GPT Image2 / Nano Banana Pro / Seedream5.0 等主流生图模型。

---

## ⚠️ 重要：使用前必读

本 Skill 生成的 **图片提示词** 是为 AI 文生图大模型设计的。将提示词复制到生图模型时，**必须同时上传以下两张参考图**，否则无法生成匹配品牌视觉的图片：

| 参考图 | 说明 | 用途 |
|--------|------|------|
| **参考图1** | BORED WHISKY 产品瓶身照片 | 生图模型据此还原产品外观、瓶身比例、标签细节 |
| **参考图2** | BORED WHISKY Logo 图片 | 生图模型据此在画面中植入品牌标识 |

> **简单理解：** Skill 输出的是"文字指令"，你把这段指令 + 两张参考图一起喂给 AI 作图工具，就能生成带有 BORED WHISKY 产品和品牌元素的小红书海报图片。

---

## 功能特点

- **内置品牌 DNA**：口号、IP、规格、瓶身比例、用户画像、使用场景全部内置
- **6+2 模板体系**：6 种结构化爆款模板 + 2 种自由创意模式
- **一问一答交互**：5 步流程，轻松生成专业级图文
- **双份输出**：图片提示词 + 小红书文案，一次搞定
- **AI 审核规避**：人物不露正脸，侧脸/背影/道具遮挡/景深虚化
- **开源字体**：所有图片文字使用开源免费可商用字体
- **潮流设计**：内置小红书爆款海报视觉规范（手写体、错落排版、POP装饰）

## 安装方式

### 方法一：直接复制（通用）

1. 克隆或下载本仓库
2. 将 `bored-whisky-xhs` 目录复制到你的 AI 平台 skills 目录：
   ```
   cp -R bored-whisky-xhs ~/.trae/skills/
   ```
3. 重启 AI 平台即可使用

### 方法二：Git 子模块（推荐）

```bash
cd ~/.trae/skills/
git submodule add https://github.com/ifeihong/bored-whisky-xhs.git
```

## 使用方法

安装后，在对话中输入：

```
$bored-whisky-xhs
```

或直接描述需求：

```
用 bored-whisky-xhs 帮我做一篇露营主题的小红书图文
```

### 交互流程

1. **输入主题** — 今天想发什么内容？
2. **选择模式** — 结构化模板 / 自由创意 / 参考图复刻
3. **选择模板**（模式1）— 6 种爆款模板任选
4. **选择情绪** — 潮酷叛逆 / 精致松弛 / 幽默玩梗 / 温暖治愈 / 高级质感
5. **确认标题** — AI 生成 3 个候选标题，你选或自己写

### 输出内容

- **图片生成提示词**：可直接复制到 GPT Image2 / Nano Banana Pro / Seedream5.0（**记得同时上传产品图和 Logo 作为参考图**）
- **小红书文案**：标题 + 多行短句正文（600字左右，配丰富 emoji）+ 分层标签
